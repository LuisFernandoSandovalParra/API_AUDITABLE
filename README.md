# API_Auditable
API-Rest basica para auditar.

Tecnologias usadas:
 - Express.js
 - Node.js
 - MariaDB
 
 Esta API cuenta con una base de datos basica, compuesta por:
 - Tablas:
  - Users
  - students
  - careers
  - students_logs
  - careers_logs
  
  Adicionalmente, la API cuenta hace uso de:
  - JWT (Tokens de seguridad)
  - Winston (Logs)
